programacao-simplificada
Web app para leitura da programação semanal de manutenção da Arlanxeo ESBR - Triunfo/RS.
O que faz
Todo domingo o time de manutenção recebe um Excel com as ordens da semana. O app permite carregar esse arquivo e visualizar as tarefas filtradas por equipe ou área, sem precisar abrir a planilha.
Como usar

Acesse o app pelo GitHub Pages
Carregue o arquivo .xlsx da semana
Filtre pela sua equipe

https://brunosteinbruck.github.io/programacao-simplificada/
Tecnologias

HTML, CSS, JavaScript
SheetJS para leitura do Excel
