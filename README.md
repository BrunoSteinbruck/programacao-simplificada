programacao-simplificada
Web app para leitura da programação semanal de manutenção da Arlanxeo ESBR - Triunfo/RS.

Todo domingo o time de operação recebe um Excel com as ordens da semana. O app permite carregar esse arquivo e visualizar as tarefas filtradas por equipamento, área e data, sem precisar abrir a planilha que é extensa e confusa.

Como usar

Acesse o app pelo GitHub Pages
Carregue o arquivo .xlsx da semana
Filtre pelas áreas e data

https://brunosteinbruck.github.io/programacao-simplificada/


Tecnologias:
HTML, CSS, JavaScript
SheetJS para leitura do Excel
